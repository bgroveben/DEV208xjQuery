In order for ajax calls to be made, you need to have a server running and be sure you are making HTTP (not HTTPS) calls. From your working directory, start the server and go to http://localhost:port/filename.html.

The most basic Ajax operation we can perform using jQuery is .get(). In this example, we will contact the target url and pass the string that the server returns into the parameter we use for our event handler. That sounds more complicated than it actually is.
