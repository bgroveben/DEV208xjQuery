Wrapping functions

As we saw earlier, jQuery provides three separate functions for wrapping existing content with new elements. The difference between the three may appear to be subtle, but will have a large impact on the resulting DOM. These demos are designed to show you the wrapping functions in action and the differences between them.

.wrap()

.wrap() will surround each element with the new content. Notice when you click the button below that each div.demo element is surrounded with a new section element.

$('div.demo').wrap('<section />');

.wrapAll()

.wrapAll() will surround the entire group of returned elements with the new content. Notice when you click the button below that all of the div.demo elements are surrounded with one new section element.

$('.demo').wrapAll('<section />')

.wrapInner()

.wrapInner() will open each of the returned elements, find their children, and surround each of the children with the new content. In this example, we will wrap all of the div elements inside of #parent with a new section.

$('#parent').wrapInner('<section />')
