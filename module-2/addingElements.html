DOM Manipulation Demonstration

Below is an interactive exercise that will demonstrate before, after, append, prepend, and their associated functions.

To use the interface, type the new element you wish to create in New element (with the starting and ending tags, and whatever ID an attributes you wish to give it). Make sure you use double quotes around the attribute values, not single quotes.

Then, type the ID of the element you wish to target in Target id. All of the buttons will reflect the code that they will execute.

To execute the code, simply click the button. The HTML is at the bottom (you have to scroll down to see it). The starting HTML consists of one div element with an id of starter. The HTML will continue to update throughout the demo, allowing you to test the various functions and see how to create the HTML you will need to use in your own pages.

Some Notes

When you look at the JavaScript for this demo, you will notice that it is NOT the straight method calls that are being demonstrated. The reason for that is to make writing the demo easier as well as to ensure that the buttons stay dynamic and the HTML remains 'prettified'. However, the code that is on the buttons is the actual code that is executed.
