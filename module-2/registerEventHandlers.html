Event, delegation, and one demonstrations

Each row contains a demonstration.

On the left you will see the instructions to walk through the demonstration. Notice that you will click on the code on the left to execute it. Once the code has been executed, it will turn yellow.

On the right, you will see the 'targets'. This is where the buttons you will be adding the click event s to will be displayed, as well as a simple output window. Each button will show its ID and any classes.

The buttons on the right are not wired up when the page is loaded, meaning the code in the samples has not been executed.

The appendDisplay helper function finds the clicked item, then finds the ol with a class of display closest to that button, and then adds the text "Clicked".

.on() WITHOUT delegation sample

1. Register event handler for all elements with on-no-delegation class by clicking the code below
            $('.on-no-delegation').on('click', appendDisplay);
2. Click on the right. Notice the text "Clicked" is displayed.
3. Remove the text and create a new button by clicking the code below.
            $('on-no-delegation').after('<button class="btn btn-info on-no-delegation new"></button>');
4. Click the second button (the one at the bottom). Notice that there is no new text displayed. Clicking on the first button still works.

    .on() WITH delegation sample

    1. Register event handler for all elements with on-delegation class by clicking the code below. Notice the change in the .on() function call. Also notice that we are using document as the base. You could use a different parent if it is appropriate.
                $(document).on('click', '.on-delegation', appendDisplay);
    2. Click the button on the right. Notice the text "Clicked" is displayed.
    Remove the text and create a new button by clicking the code below.
                $('.on-delegation').after('<button class="btn btn-info on-delegation new"></button>');
    4. click the second button (the one at the bottom). Notice that the new text is displayed.

      .one() sample

      1. Register event handler for all elements with one class by clicking the code below.
                  $('.one').one('click', appendDisplay);
      2. Click the button on the right. Notice the text "Clicked" is displayed. Notice that if you click the button again no new text is displayed.
