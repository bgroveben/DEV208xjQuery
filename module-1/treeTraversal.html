jQuery Tree Traversal Methods

A demonstration of some common jQuery tree traversal methods.

As you can see, the buttons have the jQuery code that will be executed when the buttons are clicked. The div element with the ID of target will be the starting point for all of the demos. The flag class is simply used as a demo class to show off the prevUntil and nextUntil methods.

Next and previous

prev() and next() move to either the prior or the next element, respectively.

$('#target').prev().addClass('highlight');
$('#target').next().addClass('highlight');

prevAll and nextAll

prevAll() and nextAll() will select all siblings (elements on the same level of the DOM hierarchy) prior to or after the current element, respectively.

$('#target').prevAll().addClass('highlight');
$('#target').nextAll().addClass('highlight');

prevUntil and nextUntil

prevUntil and nextUntil will select all elements before or after the current element, up to but not including the element that matches the selector passed into the method.

$('#target').prevUntil('.flag').addClass('highlight');
$('#target').nextUntil('.flag').addClass('highlight');
