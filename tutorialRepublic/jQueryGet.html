For this demo to work, you need to fire up your server and open this file on your localhost at whichever port. I use the Mac's default PHP server, so I navigate to the directory that this file is in, open a separate tab in the console, and type

[=> php -S localhost:8000

Now, go to your browser's search bar and enter http://localhost:8000/your-filename.html

You must open the demo file using http:// (NOT https) since Ajax makes HTTP requests.

Did you get all of that? If you are still having issues, I found the links here and here to be helpful. You can also do a Google search for 'jQuery Ajax' and 'jQuery get'.

 

All right, on to the demo.
